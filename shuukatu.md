# ポートフォリオアプリケーション - 就活用資料

## 1. FlagQuiz（国旗クイズ）

### 概要
世界中の国旗を楽しく学べるクイズアプリ。ゲーム感覚で地理の知識を深めることを目的としたiOSアプリケーションです。難易度別の問題モード、スコア記録機能、学習モード、タイムアタックモードなど、多様な学習スタイルに対応しています。

### 役割
- アプリの企画・設計・開発を一人で担当
- UI/UXデザインの設計と実装
- ゲームロジックの実装（クイズ機能、スコア管理、難易度設定）
- App Storeへの公開と運用

### 努力した点
- ユーザーが継続的に学習できるよう、複数のモード（学習モード、タイムアタックモード）を実装し、ゲーム性と教育性のバランスを追求
- スコア記録機能により、ユーザーの成長を可視化し、モチベーション維持を支援
- 直感的なUI設計により、年齢を問わず使いやすいインターフェースを実現
- App Store審査を通過し、実際にリリースまで完了

## 2. ThemeJinro（ホンネ人狼）

### 概要
お題付き人狼ゲームアプリ。友達との集まりをもっと盛り上げることを目的としたiOSアプリケーションです。通信ゼロで1台のスマホだけで完結する「パス&プレイ」型のオフライン人狼ゲームとして設計されています。お題×推理でトークが弾む仕組みを実現しています。

### 役割
- アプリの企画・設計・開発を一人で担当
- ゲームロジックの実装（人狼ゲームのルール、フェーズ管理）
- Firebase AuthenticationとFirestoreを使用したデータ管理
- オフライン対応の実装

### 努力した点
- 通信環境がなくても楽しめるよう、1台のスマホだけで完結する「パス&プレイ」型の設計を実現
- お題機能により、従来の人狼ゲームに新しい楽しみを追加し、トークが弾む仕組みを構築
- フェーズごとに色や効果音で切り替わる直感的な操作を実装し、迷わないスムーズな体験を提供
- Firebaseを活用した認証とデータ管理により、セキュアなアプリケーションを実現
- App Store審査を通過し、実際にリリースまで完了

## 3. YataiMobile（屋台モバイル）

### 概要
イベント運営支援アプリ。スマホがハンディに変わる、屋台や学園祭での注文管理・売上集計を効率化することを目的としたiOSアプリケーションです。キッチン管理機能、売上追跡・集計、オプションのカスタマイズ、ハンディ機能、グループ番号共有による複数スタッフとの協力、注文番号管理、利用者集計機能などを提供します。

### 役割
- アプリの企画・設計・開発を一人で担当
- イベント運営の業務フローを分析し、必要な機能を設計
- Firebase（Firestore）を使用したリアルタイムデータ同期機能の実装
- 複数スタッフ間でのデータ共有機能の実装
- ハンディ機能の実装（移動中でも操作可能）

### 努力した点
- 実際のイベント運営現場でのニーズを調査し、キッチン管理、売上集計、注文管理など、実用的な機能を網羅的に実装
- グループ番号共有機能により、複数のスタッフがリアルタイムでデータを共有できる仕組みを構築
- Firebase Firestoreを活用したリアルタイム同期により、複数デバイス間でのデータ整合性を実現
- ハンディ機能により、移動中でも簡単に操作できるよう、UI/UXを最適化
- 大規模なイベントでもスムーズに運営できるよう、パフォーマンスを考慮した設計
- App Store審査を通過し、実際にリリースまで完了

## 4. カチカン（価値換）

### 概要
日々の予算をカテゴリーごとに設定し、買い物の価値を時間軸で実感する支出管理アプリ。従来の支出管理アプリでは「使いすぎ」は分かっても、購入の「意思決定」そのものを支援しにくいという課題を解決することを目的としたiOSアプリケーションです。

### 役割
- アプリの企画・設計・開発を一人で担当
- ユーザーの消費行動の課題を分析し、解決策を設計
- 予算の自動積立機能の実装
- ウィッシュリスト機能の実装（目標金額達成までの日数計算）
- ダッシュボード/分析機能の実装（グラフ可視化）

### 努力した点
- 従来の支出管理アプリの課題（「使いすぎ」は分かるが「意思決定」を支援しにくい）を分析し、時間軸で価値を実感できる新しいアプローチを提案
- 予算の自動積立機能により、毎日貯まっていくプロセスを可視化し、ゲーム感覚で楽しく節約できる仕組みを実現
- 「この買い物は、あなたの〇〇日分の予算です」というフィードバックにより、モノの価値を時間軸で実感できるよう設計
- ウィッシュリスト機能により、欲しい商品を登録し、目標金額達成までの日数を自動計算することで、モチベーション維持を支援
- ダッシュボード/分析機能により、どのカテゴリーにどれだけの「日数分の価値」を使っているかをグラフで可視化し、支出パターンを把握しやすく
- 節約や予算管理が「我慢」ではなく、楽しく継続できるようなUX設計を追求
- App Store審査を通過し、実際にリリースまで完了

## ポートフォリオサイト

### 概要
自作アプリケーションを紹介するポートフォリオサイト。React + TypeScript + Viteで構築され、GitHub Pagesで静的サイトとして公開されています。

### 役割
- サイトの企画・設計・開発を一人で担当
- React + TypeScript + Viteを使用したモダンなフロントエンド開発
- Tailwind CSSを使用したUIデザイン
- GitHub Pagesへのデプロイ設定

### 努力した点
- React + TypeScript + Viteを使用したモダンな開発環境の構築
- Tailwind CSSを活用し、保守性の高いスタイリングを実現
- 各アプリのテーマカラーを考慮したデザイン設計
- レスポンシブデザインにより、様々なデバイスに対応
- GitHub Pagesでの静的サイトホスティングにより、運用コストを抑えた構成
- 各アプリの詳細ページで、概要、機能、プライバシーポリシーなどを分かりやすく表示
- AIにありがちなUIを避け、落ち着いた信頼感のあるデザインを追求
