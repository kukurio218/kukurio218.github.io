ポートフォリオサイトプロジェクト

概要
このプロジェクトは僕のポートフォリオサイトです。

技術スタック
* React(TypeScript): プロジェクト作成と再構築に使用。
* Tailwind CSS: UIデザインに使用。
* Github Pages: 静的書き出しを有効化（重要）。

UIデザインガイドライン
* AIにありがちなUIは避けてください。
* 参考資料：
    * [Improving Frontend Design Through Skills](https://claude.com/blog/improving-frontend-design-through-skills)
    * [UI/UX Pro Max Skill](https://ui-ux-pro-max-skill.nextlevelbuilder.io/)

ベストプラクティス
* Reactのベストプラクティスを活用するために、以下のコマンドを実行してください。
    * `$ npx add-skill vercel-labs/agent-skills`

目標
* それぞれのアプリが魅力的に見えるようにしてください。
* 紹介ページも魅力的である必要があります。
* 各アプリのテーマカラーは、同じでも違くても、魅力的であれば構いません。
