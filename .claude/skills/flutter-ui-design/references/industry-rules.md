# Industry-Specific Design Rules for Mobile Apps

100の業種別デザインルール。ユーザーの業種を聞いたら、このルールを参照して最適なデザインシステムを提案する。

---

## 使い方

1. ユーザーの業種/アプリ種類を特定
2. 該当するルールを参照
3. 推奨スタイル、カラームード、フォントムード、避けるべきパターンを適用
4. Pre-delivery Checklistで品質確認

---

## Tech & SaaS（15ルール）

### 1. SaaS / B2B Tool
- **推奨スタイル:** Minimalism, Swiss Style, Bento Grid
- **カラームード:** Professional Blue, Neutral Gray, Subtle Accent
- **フォントムード:** Clean Sans-serif (Inter, DM Sans)
- **キーエフェクト:** Subtle hover states, Smooth transitions (200ms)
- **避ける:** 派手な色、過剰なアニメーション、カジュアルすぎるトーン

### 2. Developer Tool / IDE
- **推奨スタイル:** Dark Mode (OLED), Minimalism, Monospace Focus
- **カラームード:** Dark background, Syntax highlighting colors
- **フォントムード:** Monospace (JetBrains Mono, Fira Code)
- **キーエフェクト:** Fast feedback (<100ms), Keyboard-first
- **避ける:** ライトモードのみ、遅いアニメーション

### 3. AI / Chatbot Platform
- **推奨スタイル:** AI-Native UI, Glassmorphism, Motion-Driven
- **カラームード:** Gradient accents, Pulsing effects, Dynamic
- **フォントムード:** Modern Sans (SF Pro, Inter)
- **キーエフェクト:** Typing indicators, Thinking animations, Streaming text
- **避ける:** 静的すぎるUI、古いチャットUI

### 4. Analytics Dashboard
- **推奨スタイル:** Data-Dense, Bento Grid, Dark Mode
- **カラームード:** Neutral base, Data visualization colors
- **フォントムード:** Tabular numerals (Roboto Mono for numbers)
- **キーエフェクト:** Chart animations, Drill-down transitions
- **避ける:** 装飾的な要素、データより見た目優先

### 5. Productivity / Task Management
- **推奨スタイル:** Minimalism, Flat Design, Micro-interactions
- **カラームード:** Clean white/gray, Accent for priorities
- **フォントムード:** Readable Sans (Nunito, Open Sans)
- **キーエフェクト:** Checkbox animations, Drag & drop feedback
- **避ける:** 複雑なUI、気が散る装飾

```
│  TARGET: [アプリ名] - RECOMMENDED DESIGN SYSTEM                 │
├─────────────────────────────────────────────────────────────────┤
│  INDUSTRY: [業種]                                               │
│  STYLE: [推奨スタイル]                                          │
│                                                                 │
│  COLORS:                                                        │
│     Primary:    #XXXXXX ([名前])                                │
│     Secondary:  #XXXXXX ([名前])                                │
│     Background: #XXXXXX ([名前])                                │
│     Text:       #XXXXXX ([名前])                                │
│     Accent:     #XXXXXX ([名前])                                │
│                                                                 │
│  TYPOGRAPHY:                                                    │
│     Display: [フォント名]                                       │
│     Body: [フォント名]                                          │
│     Google Fonts: [URL]                                         │
│                                                                 │
│  KEY EFFECTS:                                                   │
│     [効果1] + [効果2] + [効果3]                                 │
│                                                                 │
│  AVOID (Anti-patterns):                                         │
│     [避けるべきパターン]                                        │
│                                                                 │
│  PRE-DELIVERY CHECKLIST: → references/pre-delivery-checklist.md │
└─────────────────────────────────────────────────────────────────┘
```
