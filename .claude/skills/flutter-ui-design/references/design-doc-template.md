## 使用方法

1. 壁打ちの質問が全て完了したら、このテンプレートに沿ってドキュメントを生成
2. 決まっていない項目は `[TBD]` または `[要検討]` と記載
3. 必要に応じてセクションを追加・削除
4. ユーザーに確認してもらい、必要なら修正

## 出力時の注意

- コードブロックで囲まず、Markdownとして出力
- 実際の値を埋める（テンプレートの `[xxx]` を置き換える）
- 図は可能な限りテキストで表現（Mermaidは使わない）
