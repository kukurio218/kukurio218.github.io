# Claude Code Permissions

Claude is allowed to:

- Read all files in the repository
- Create new files
- Modify existing files
- Delete files
- Execute shell commands
- Install and remove npm packages
- Run build and dev commands
- Refactor code across multiple files
- Create and modify configuration files
- Generate documentation

Claude is NOT allowed to:

- Access files outside this repository
- Upload data to external services
- Modify environment variables outside this project
- Execute destructive system-wide commands
